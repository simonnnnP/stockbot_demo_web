# Demo web StockBot

## Requisitos
- Python 3.10+
- pip install flask

## Cómo ejecutar
```bash
export FLASK_APP=app.py
flask run
```

Abre http://127.0.0.1:5000 en tu navegador.

## Notas
- Los datos están en memoria y se pierden al reiniciar.
- Para conectarlo a Excel usa Microsoft Graph, igual que el bot de WhatsApp.
